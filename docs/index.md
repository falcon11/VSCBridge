## Getting Started

Create first doc in manual way

```bash
// Create dir for libraries
$ mkdir library && cd library

// Install dumi
$ npm i dumi

// Create docs
$ mkdir docs && echo '# Hello dumi!' > docs/index.md

// Preview docs
$ npx dumi dev
```
